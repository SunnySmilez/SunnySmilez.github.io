php.ini基本参数设置

发表于 2016-07-05 | 分类于 PHP |

开启安全模式(阻止php调用部分函数,比如system())

safe_mode = On

禁用函数

disable_functions=system,exec,shell_exec

关闭php版本信息在header信息中泄露

expose_php = Off

关闭注册全局变量

register_globals = Off

打开magic_quotes_gpcf防止sql注入(自动将’转换成\’)

magic_quotes_gpc = On

错误消息控制(不在页面显示)

display_errors = Off
error_reporting = E_WARNING & E_ERROR

错误日志

log_errors = On
error_log = /logs/php_error.log

设置脚本运行最长时间

max_execution_time = 30

每个脚本使用的最大内存

memory_limit = 128M

每个脚本等待输入数据最长时间

max_input_time = 60

上传文件的最大的大小

upload_max_filesize= 2M
